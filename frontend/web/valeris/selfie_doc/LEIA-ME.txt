
VALERIS KYC COMPONENT - SELFIE_DOC
==========================================

Para usar este componente:
1. Abra o arquivo selfie_doc.html
2. Procure pela chamada ValerisSDK.init no final do arquivo
3. Insira seu TOKEN e a URL da sua API

Configuração de exemplo:
ValerisSDK.init({
    apiUrl: "https://sua-api.com/api",
    token: "SEU_TOKEN_AQUI"
});

O componente já vem configurado para funcionar de forma independente.
